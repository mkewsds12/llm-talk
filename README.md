# 大模型入门：从概念到实战

一个面向教学与分享场景的静态演示项目，内容覆盖大模型基础概念、Transformer、Agent、RAG、MCP、常见开发资源与安全风险等主题。

## 项目说明

- 这是一个纯静态项目，可直接通过浏览器打开 `index.html` 预览。
- 项目主要用于教学演示、知识分享、学习交流与非商业展示。
- 项目中的页面内容、结构设计、资源整理与示例代码均服务于教学目的。

## 目录结构

- `index.html`：演示主页面
- `css/style.css`：页面样式
- `js/main.js`：交互脚本
- `assets/`：图标、插图等静态资源

## 使用方式

### 本地预览

直接打开 `index.html` 即可。

### 静态部署

可部署到任意静态托管平台，例如：

- GitHub Pages
- Cloudflare Pages
- Vercel
- Netlify
